1. Download the Addon (애드온을 다운 받으세요)
Check the file, easy_moduleinstaller_ko.py
다운 받은 easy_moduleinstaller_ko.py 파일을 확인하세요


2. Install in Blender (블렌더에서 설치)
Open Blender, and Go to Edit > Preferences.
(블렌더를 실행하고 편집 > 환경설정 으로 이동하세요)

Select the Add-ons tab on the left.
(왼쪽의 애드온 탭을 선택 하세요)

Click the Install from Disk... button at the top right.
(우측 상단의 Install from Disk... 버튼을 누르세요)

Locate the downloaded easy_module_installer_ko.py file and click "Install Add-on".
(다운 받았던 easy_moduleinstaller_ko.py 파일을 선택하고 "Install Add-on"을 클릭하세요)

Enable the addon by checking the box next to Development: Module Installer.
(설치 후 체크박스를 활성화 해 애드온을 활성화 해주세요)

3. How to Use (Install & Uninstall) (사용법(설치 & 삭제))
a. Press N in 3D Viewport to open Sidebar.
  (뷰포트에서 N을 눌러 사이드바를 열어주세요)

b. Type in the module name you want in the placeholder below "+ Install New Package:"
   ("+ Install New Package:" 밑의 입력창에 원하는 모듈을 입력하세요)

c. Install: click Install Package.
   (설치: Install Package를 클릭하세요)

d. Uninstall: Select a package from the list and click Uninstall Selected.
   (삭제: Installed Packages 리스트에서 삭제하고 싶은 모듈을 클릭하고 아래 Uninstall Selected를 클릭하세요)

e. Click the refresh icon if the list is empty.
   (삭제 후 새로고침 아이콘을 눌러 리스트를 업데이트 하세요)
